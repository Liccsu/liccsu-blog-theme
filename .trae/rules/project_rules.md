# Sky Blog Theme - 项目规则

## 📋 项目概述

**Sky Blog Theme** 是基于现代前端技术栈的 Halo 2.0 主题，集成 Vite、Tailwind CSS v4、DaisyUI 和 Alpine.js，提供优雅的博客体验。

### 主题特色
- **现代化设计**: 基于 Tailwind CSS v4 和 DaisyUI 的现代化 UI 设计
- **响应式布局**: 完美适配桌面端和移动端
- **主题切换**: 支持深色/浅色模式切换
- **性能优化**: 基于 Vite 的快速构建和热更新
- **类型安全**: 完整的 TypeScript 支持
- **组件化**: 基于 Alpine.js 的轻量级交互组件

## 🛠️ 环境要求

### 必需环境
- **Java**: 21+ (Halo 运行环境)
- **Node.js**: 20+ (前端开发环境)
- **pnpm**: 10.6.5+ (包管理器)
- **Halo**: 2.21+ (博客系统)

### 开发工具
- **IDE**: 推荐使用 VS Code 或 WebStorm
- **Git**: 版本控制
- **终端**: 支持现代 shell 环境

## 🏗️ 技术栈规范

### 核心技术栈
- **构建工具**: Vite 7.0.3+ (快速构建和热更新)

- **后端框架**: Halo 2.21.3+ (博客系统)
