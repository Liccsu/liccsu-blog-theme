# 首页设置 (Index)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 首页**

配置首页的各个模块和布局。

---

## 1. 首页头部区域

| 设置项 | 说明 |
|--------|------|
| **启用首页头部** | 关闭后隐藏整个头部区域 |

---

## 2. 头部背景设置

| 设置项 | 说明 |
|--------|------|
| **背景类型** | `背景图片` 或 `CSS 特效` |
| **背景媒体** | 上传图片/视频或输入 URL |
| **特效样式** | 流体渐变 / 霓虹线条 / 流星雨 / 深邃星空 |

---

## 3. 蒙版渐变设置

仅在使用背景图片时显示。

| 设置项 | 说明 |
|--------|------|
| **浅色主题蒙版** | 黑色 / 白色 / 主题背景色 / 自定义 |
| **深色主题蒙版** | 黑色 / 白色 / 主题背景色 / 自定义 |

---

## 4. 主标题设置

| 设置项 | 说明 |
|--------|------|
| **显示主标题** | 是否显示 |
| **自定义主标题** | 留空使用站点标题 |
| **字体** | 无 / Monoton / Titan One / 霞鹜文楷亮 / 思源宋 |
| **样式效果** | 无 / 麦克笔感 / 斜切双色 / 流光渐变 / 描边发光 / 微光晕染 / 简约阴影 |
| **装饰效果** | 无 / 粒子轨道 / 星光闪烁 |

---

## 5. 副标题设置

| 设置项 | 说明 |
|--------|------|
| **显示副标题** | 是否显示 |
| **副标题文本** | 副标题内容 |
| **字体** | 同主标题 |
| **样式效果** | 无 / 打字机 / 渐变流动 / 微发光 / 呼吸 |

---

## 6. 底部过渡设置

| 设置项 | 说明 |
|--------|------|
| **显示波浪过渡** | 是否显示波浪过渡效果 |
| **波浪颜色** | 波浪的颜色值 |

---

## 7. 瞬间说说设置

| 设置项 | 说明 |
|--------|------|
| **显示瞬间说说** | 是否在头部显示瞬间模块 |
| **瞬间标题** | 模块标题文字 |

---

## 8. 侧边栏设置

| 设置项 | 说明 |
|--------|------|
| **显示侧边栏** | 是否显示 |
| **定制侧边栏** | 启用后可单独配置首页侧边栏 |
| **侧边栏位置** | 左侧 / 右侧 |
| **小工具列表** | 拖拽排序小工具 |

---

## 9. 首页内容设置

| 设置项 | 说明 |
|--------|------|
| **显示热力图** | 是否显示热力图 |
| **热力图类型** | GitHub 贡献 / 文章发布 |
| **Tab 切换风格** | 药丸 / 芯片 / 下划线 / 玻璃拟态 |
| **Tab 排序** | 拖拽调整首页 Tab (全、瞬间、友链等) 的显示顺序 |
| **显示文章列表** | 是否显示“全部文章” Tab 及其内容 |

---

## 10. 文章内容设置

| 设置项 | 说明 |
|--------|------|
| **文章列表风格** | 卡片 / 列表 / 杂志 / 极简 |
| **文章数量** | 首页显示数量（3-12） |
| **启用封面替代** | 无封面时使用默认图片 |
| **替代模式** | 无封面时替代 / 始终使用 |
| **替代图片地址** | 默认封面图片 URL |

---

## 11. 朋友动态设置

需安装 `plugin-friends` 插件。

| 设置项 | 说明 |
|--------|------|
| **显示朋友动态** | 是否显示此 Tab |
| **朋友动态数量** | 显示数量（3-20） |
| **列表样式** | 两列 / 单列 |

---

## 12. 瞬间说说设置

需安装 `plugin-moments` 插件。

| 设置项 | 说明 |
|--------|------|
| **显示瞬间说说** | 是否显示此 Tab |
| **瞬间数量** | 显示数量（3-12） |
| **列表样式** | 两列 / 单列 |

---

## 13. 图库模块

需安装 `plugin-photos` 插件。

| 设置项 | 说明 |
|--------|------|
| **启用图库模块** | 是否显示 |
| **模块标题** | 标题文字 |
| **模块副标题** | 副标题文字 |

---

## 14. 友情链接模块

需安装 `plugin-links` 插件。

| 设置项 | 说明 |
|--------|------|
| **启用友情链接** | 是否显示 |
| **模块标题** | 标题文字 |
| **模块副标题** | 副标题文字 |

---

## 15. 悬浮控制栏

| 设置项 | 说明 |
|--------|------|
| **定制悬浮控制栏** | 启用后可单独配置 |
| **Dock 菜单** | 选择菜单 |
| **返回顶部按钮** | 是否显示 |
