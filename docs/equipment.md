# 装备页设置 (Equipment)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 装备**

展示你的硬件装备、软件工具或任何你想分享的好物。需配合 `plugin-equipment` 插件使用（如果存在插件版）或直接在主题配置中添加数据（视实现而定）。

---

## 1. 基础信息

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **页面标题** | 装备页面的标题 | "我的装备" |
| **页面描述** | 装备页面的描述文字 | "分享我日常使用的硬件与软件" |

---

## 2. 风格设置

| 设置项 | 说明 | 选项 |
|--------|------|------|
| **显示模式** | 页面展示风格 | 科技风 (Tech) / 简约风 (Simple) |

---

## 3. 功能交互

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用评论** | 在页面底部显示评论区 | 开启 |

---

## 4. 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 是否显示侧边栏 | - |
| **定制侧边栏** | 启用后可单独配置，禁用则使用全局配置 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

---

## 5. 装备数据配置

装备数据需在主题配置中的"装备分组"中手动添加。

| 设置项 | 说明 |
|--------|------|
| **分组图标** | 支持 **Iconify** 图标库，点击输入框即可弹出选择器预览并选择图标。 |
| **分组名称** | 装备类别的名称（如：主力设备、外设、软件） |
| **装备列表** | 在每个分组下添加具体的装备项 |
