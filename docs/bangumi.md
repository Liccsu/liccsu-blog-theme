# 追番页面设置 (Bangumi)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 追番页面**

需安装 Bilibili 追番插件并配置 B 站 UID。

**插件信息**：
- 应用市场：[app-OTFPN](https://www.halo.run/store/apps/app-OTFPN)
- GitHub：[plugin-bilibili-bangumi](https://github.com/Roozenlz/plugin-bilibili-bangumi)

---

## 页面设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **页面标题** | 追番页面的标题 | "追番列表" |
| **页面描述** | 追番页面的描述文字 | "记录我的二次元之旅" |

---

## 追番卡片设置

侧边栏追番轮播卡片的显示配置。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **番剧类型** | 追番 / 追剧 | 追番 (1) |
| **番剧状态** | 全部 / 想看 / 在看 / 已看完 | 在看 (2) |
| **展示数量** | 轮播卡片中显示的番剧数量 | 6 |

### 类型说明

- **追番 (1)**：动画番剧
- **追剧 (2)**：电视剧、综艺等

### 状态说明

- **全部 (0)**：显示所有状态的番剧
- **想看 (1)**：计划观看的番剧
- **在看 (2)**：正在追的番剧
- **已看完 (3)**：已完结的番剧

---

## 侧边栏小工具

在侧边栏设置中选择 **"追番卡片"**，可显示轮播样式的追番展示。

### 特性

- **轮播展示**：自动切换展示多个番剧
- **封面展示**：显示番剧封面图
- **进度显示**：显示观看进度
- **评分显示**：显示个人评分
- **链接跳转**：点击跳转到详情页

---

## 插件配置

> 路径：**后台 → 插件 → Bilibili 追番**

### 必需配置

| 配置项 | 说明 |
|--------|------|
| **B 站 UID** | 你的 Bilibili 用户 ID |
| **同步间隔** | 自动同步追番数据的时间间隔 |

### 获取 B 站 UID

1. 登录 Bilibili
2. 进入个人空间
3. URL 中的数字即为 UID
   - 例如：`https://space.bilibili.com/123456` → UID 为 `123456`

## 注意事项

1. **插件依赖**：必须安装并启用 `plugin-bangumi` 插件
2. **数据同步**：首次配置后需要等待插件同步数据
3. **API 限制**：B 站 API 可能有访问频率限制
4. **隐私设置**：确保 B 站账号的追番列表为公开
