# 认证页面设置 (Auth)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 认证页面**

配置登录、注册等认证页面的样式。

---

## 页面布局

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **页面布局** | 系统默认 / 分屏布局 / 居中卡片 / 极客终端 | 分屏布局 |

### 布局说明

| 布局 | 说明 |
|------|------|
| **系统默认** | 使用 Halo 默认的登录页面 |
| **分屏布局** | 左侧品牌展示区 + 右侧登录表单 |
| **居中卡片** | 表单居中显示 |
| **极客终端** | 终端风格界面 |

---

## 分屏布局设置

仅在选择"分屏布局"时显示。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **品牌区样式** | 简约品牌 / 功能特性 | 简约品牌 |
| **品牌标语** | 在品牌展示区显示的标语文字 | "探索、记录、分享" |

### 简约品牌风格

| 设置项 | 说明 |
|--------|------|
| **站点标题** | 简约风格下的站点标题，留空使用站点默认标题 |

### 功能特性风格

| 设置项 | 说明 |
|--------|------|
| **Logo** | 留空使用站点默认 Logo |
| **标题** | 留空使用站点默认标题 |
| **侧边栏背景图** | 自定义背景图片，覆盖默认渐变 |
| **欢迎语** | 默认 "欢迎来到" |
| **特性 1/2/3** | 图标 + 标题 + 描述 |

默认特性：
- 🛡️ 安全可靠 - 基于 Spring Security 的账户保护
- ✨ 内容推荐 - 为您量身定制的阅读体验
- 📱 多端同步 - 随时随地访问您的创作

---

## 居中卡片布局设置

仅在选择"居中卡片"时显示。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **卡片效果** | 3D 倾斜 / 聚光灯 | 3D 倾斜 |
